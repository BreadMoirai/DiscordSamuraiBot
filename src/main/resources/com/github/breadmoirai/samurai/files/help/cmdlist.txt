**--- General ---**
`[prefix]help....` -displays help. Mentioning the bot will also display help.
`[prefix]join....` -invite link for DreadMoirai's Samurais.
`[prefix]invite..` -link to add Samurai to your server.

**--- Games  ---**
`[prefix]duel....` -starts a connectFour game.
`[prefix]hangman.` -starts a hangman game.
`[prefix]roll....` -Rolls a dice

**--- Music  ---**
use `[prefix]help music` and see #how-to-music

**--- Manage ---**
`[prefix]prefix..` -changes the command prefix.
`[prefix]module..` -modules. use `[prefix]help` module for more info
`[prefix]alias...` -create an alias

**--- Notes ---**
This bot is actively being developed.
`[prefix]suggest.` -suggest a new feature.
`[prefix]enhance.` -suggest an improvement for a feature.
`[prefix]fix.....` -report a bug. Include relevant command.
Please provide details with the above commands.
