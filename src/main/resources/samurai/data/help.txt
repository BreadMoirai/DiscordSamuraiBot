**--- Generic ---**
`[prefix]help..` -displays help. Mentioning the bot will also display help
`[prefix]guild.` -invite link for DreadMoirai's Samurais.
`[prefix]invite` -link to add Samurai to your server.
`[prefix]duel..` -starts a connectFour game. ?`@mention`
`[prefix]stat..` -displays users stats.  ?`@mention`

**--- Osu! ---**
`[prefix]upload` -Must be accompanied by a uploaded `scores.db` file
`[prefix]setosu` -Links discord account to osu profile for tracking `username`
`[prefix]getosu` -get profile information. ?`username` ?`@mention`

**--- Bot ---**
`[prefix]prefix` -changes the command prefix. `prefix`[limit=4chars]
`[prefix]status` -displays status of bot
`[prefix]uptime` -displays running time of bot
`[prefix]update` -displays latest version changelog

**--- Notes ---**
Ex. `[prefix]duel @nemesis#1256` or `[prefix]duel`
At the end of a command, a `argument` may be specifed
`?` indicates that it is optional, otherwise required
