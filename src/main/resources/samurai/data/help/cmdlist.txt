**--- General ---**
`[prefix]help....` -displays help. Mentioning the bot will also display help
`[prefix]join....` -invite link for DreadMoirai's Samurais.
`[prefix]invite..` -link to add Samurai to your server.
`[prefix]duel....` -starts a connectFour game. ?`@mention`
`[prefix]perm....` -shows list of permissions. Red = missing permission

**--- Osu! ---**
`[prefix]link...` -Links discord account to osu profile for tracking `username`
`[prefix]rank...` -Displays a list of users in guild by local ranking
`[prefix]info...` -displays Samurai's info for user.  ?`@mention`
`[prefix]profile.` -displays Osu! profile ?`osu!username` ?`@mention`
`[prefix]upload..` -Must be accompanied by a uploaded `scores.db` file

**--- Bot ---**
`[prefix]prefix..` -changes the command prefix. `prefix`
`[prefix]status..` -displays status of bot
`[prefix]uptime..` -displays running time of bot

**--- Notes ---**
At the end of a command, an `argument` may be specifed
`?` indicates that it is optional, otherwise required
use `[prefix]help command` for more info on a specific command
